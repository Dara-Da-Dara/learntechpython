# Optimistic Initial Values in Reinforcement Learning

## 1. Introduction
Optimistic Initial Values is an exploration strategy used in Reinforcement Learning (RL), particularly in **value-based methods** such as multi-armed bandits and Q-learning. The main idea is to initialize action-value estimates with **intentionally high values** so that the agent is encouraged to explore all actions.

---

## 2. Definition
**Optimistic Initial Values** refers to the technique of initializing the action-value function \( Q(s, a) \) with values that are **higher than the true expected rewards**. This optimism makes untried actions appear attractive, forcing the agent to explore them.

---

## 3. Motivation
In reinforcement learning, an agent must balance:
- **Exploration**: trying new actions
- **Exploitation**: choosing the best-known action

Optimistic Initial Values promote exploration **without using randomness**, unlike Îµ-greedy methods.
